PS3 Linux has been developed on working with open source communities. If you have interested in those activities, please join.

OSS for CBE ML

It is mailing list for discussion about open source software for the Cell Broadband Engine. It treats anything about cell-generic software, including PS3 Linux kernel, other Cell Linux, libspe, and so on.

  • URL: https://ozlabs.org/mailman/listinfo/cbe-oss-dev

Linux on PowerPC developers’ ML

It is mailing list for discussion for development of Linux on PowerPC, including the kernel core PowperPC support and non-embedded platforms. Embedded platform discussions have been held at another ML.

  • URL: https://ozlabs.org/mailman/listinfo/linuxppc-dev

Toolchain Web Sites

The information about toolchain, i.e. GCC, Binutils, Gdb and Newlib, including PPE and/or SPE specific implementations, is available at the following web sites:

  • GCC: http://gcc.gnu.org/
  • Binutils: http://sourceware.org/binutils/
  • Gdb: http://sourceware.org/gdb/
  • Newlib: http://sourceware.org/newlib/

There are also a number of MLs for disucssions about the toolchain, and you can subscribe to them at the web sites above.

kboot Web Site

The kboot has a web site below for development and discussions. You can also subscribe to a ML for kboot at the web site.
  • URL: http://kboot.sourceforge.net/

CBE Specific Softwares

Available are public repositories for Cell Broadband Engine specific softwares below.

  • libspe: http://sourceforge.net/projects/libspe/
  • SIMD math library/Vector math library: http://www.continuousphysics.com/Bullet/ (included in the Bullet physics library package)

perfmon2 Web Site

The perfmon2 has a web site below for development and discussions. You can also subscribe to a ML for perfmon2 at the web site.
  • URL: http://perfmon2.sourceforge.net/
